【1단계】 GitHub 저장소 생성 (1분)

   1. 브라우저에서 https://github.com/new 접속
   2. Repository name: Playcat_ChatBot
   3. Public 선택
   4. ❌ "Add a README file" 체크 해제 (이미 있음)
   5. Create repository 클릭

【2단계】 코드 푸시 (30초)

   GitHub에 표시된 명령어 대신 아래를 사용:

   PowerShell에서:
   ───────────────────────────────────────────────
   cd c:\Users\playc\Project\Playcat_ChatBot
   git remote add origin https://github.com/YOUR_USERNAME/Playcat_ChatBot.git
   git push -u origin main
   ───────────────────────────────────────────────

   ⚠️ YOUR_USERNAME을 실제 GitHub 사용자명으로 변경!

【3단계】 Render.com 배포 (5분)

   1. https://render.com 접속
   2. "Get Started" 클릭 → GitHub 계정으로 로그인
   3. "New +" → "Web Service" 클릭
   4. "Connect a repository" → "Playcat_ChatBot" 선택
   5. 설정 확인:
      ✓ Name: playcat-chatbot-api
      ✓ Region: Singapore
      ✓ Branch: main
      ✓ Root Directory: (비워둠)
      ✓ Build Command: pip install -r requirements.txt
      ✓ Start Command: uvicorn main:app --host 0.0.0.0 --port $PORT

   6. Environment Variables (Add Environment Variable 클릭):
      ───────────────────────────────────────────────
      USE_GEMINI = true
      GEMINI_API_KEY = AIzaSyByspltgfRno_NisFdYIFZ89HCoaZdokNU
      ───────────────────────────────────────────────

   7. "Create Web Service" 클릭
   8. 배포 시작! (5-10분 소요)

      → repo 권한 선택

      환경 변수 재확인
